# 🧬 PhyN3Xs' Bioinformatics Journal 🧬

---
## 🎯 About project 🎯

This repo documents my first encounter with bioinformatics. I'm using real
biological data from **NCBI**, **UniProt** and **PDB** to learn Python 
and expand my knowledge of molecular biology at the same time.
